# ✨ Mi Glow - Self-Care TUI 💕

![Mi Glow](https://img.shields.io/badge/Made%20with%20Love-Pink-ff69b4?style=for-the-badge)

**A super cute Self-Care & Glow Up assistant** made for girls who love to shine ✨

Made with **Thugh-TUI** framework by a Venezuelan girl energy 🌴💖

---

### 🌸 Features
- Daily Skincare Routine
- Mood Checker + Affirmations
- Outfit Planner
- Beauty Products Tracker
- Glow Up Progress
- Venezuelan good vibes


### 💖 Made with
- **[Thugh-TUI](https://github.com/MRThugh/thugh-tui)** — The cutest TUI framework ever! 

**Special thanks to [@MRThugh](https://github.com/MRThugh)** for creating such an amazing pure Bash TUI framework. ¡Gracias mi rey! 💕

---

### How to run

```bash
git clone https://github.com/PixelValeria/mi-glow.git
cd mi-glow
chmod +x mi-glow.sh
./mi-glow.sh
```
